I‚Äôm using the free viridis-theme by C√©dric Scherer. It‚Äôs based on the viridis colour palette by David Cooley, which is designed for people with colour blindness.

I‚Äôm red-green colourblind and the syntax highlighting of this theme lets me distinguish easily between the different code elements. I also use syntax highlighting in the console. You can get this by checking the box at Tools > Global Options > Code > Display > Show syntax highlighting in console input.

Sometimes I switch to a light theme when writing text-rich blog posts. Syntax highlighting is less important in this instance and I find black-on-white text easier to read.

RData

There are default settings that let RStudio open from where you left off, so that the contents of your environment are intact. I don‚Äôt like this and it‚Äôs advised against in places like the R for Data Science book by Wickham and Grolemund.

I tend to restart R every few minutes so that I‚Äôm working with a completely clean environment. This stops me from modifying objects and forgetting about it, which could result in erroneous output.

You can set this behaviour in Tools > Global Options > General > Basic > Workspace by unchecking Restore .Rdata on startup and setting Save workspace to .RData on exit to Never.
