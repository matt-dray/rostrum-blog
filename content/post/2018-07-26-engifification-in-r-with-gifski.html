gifski

The `gifski’ package for R has been released. It was built by Jeroen Oom (@opencpu) for ROpenSci.

make_gif()

I’ve made a quick function for my dray package that takes a folder of PNGs and stitches them into a gif using the gifski::gifski() function. I’ve called it make_gif() because it makes gifs.

You can find the documentation on the dray package site, which is built with pkgdown.
