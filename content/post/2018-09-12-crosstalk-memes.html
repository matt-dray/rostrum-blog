Matt Dray

EARL 2018

EARL 2018 is a conference for solving business problems with R.

I’m speaking there about the Crosstalk package for the R programming language.

Crosstalk allows HTML widgets – lovely JavaScript visuals acessible in R – to interact with each other. This means that selecting points in a table will update points in a map, for example. This replicates some functions of the Shiny package – a popular R tool for interactive apps – without the need for a server or knowledge of how to program Shiny apps.

My talk describes how I helped policy makers explore schools data by creating a self-service tool incrementally with the R-Markdown-powered package Flexdashboard and Crosstalk for interactivity between the HTML widgets for interactive maps (Leaflet) and tables (DT).

Yeah, so there’s a false equivalency here. Crosstalk doesn’t necessarily help provide a direct replacement for Shiny. You still need Shiny to make ‘proper’ apps. But hey, the picture that the little girl has drawn still looks like a cat, right?

Since everything is rendered in-browser, you’re limited by your browser’s power. This means that the number of points on your interactive map, for example, is limited.
