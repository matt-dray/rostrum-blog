This is a post to help jog my memory in future, but you may find it useful.

At some point a useR will want to wrap up their reusable functions into a package, but package building can seem daunting.

I learnt how to create an R package by reading Hilary Parker’s now classic blog post for beginners.

Tom Westlake has updated the advice with references to the {usethis} package, which abstracts away some tedious setup tasks

I’m assuming you now have a minimal package with some functions and documentation and that it’s on GitHub or wherever.
