Pivot

Why? Since {tidyr} version 1.0.0, the pivot_longer() and pivot_wider() functions have replaced the spread() and gather() functions.

These functions switch data between a ‘wide’ and ‘long’ forms form. For example, a long data set might have a column called ‘year’ and a column called ‘population’. A wide version would have separate columns named for each year (‘1994’, ‘1995’, etc) that each contained that year’s ‘population’ figures.

pivot_wider() can spread the long data into the wide form and pivot_longer() does the opposite. You can learn more in the {tidyr} vignette about pivoting.
